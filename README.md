## Overview

Lead IQ analyzes a business website and basic context to estimate how strong of a lead it is for AI or automation services.

The tool outputs a simple score, key reasons, a recommended offer angle, and a suggested outreach opener.

## What It Does

- Scores leads from 0 to 100  
- Identifies key reasons behind the score  
- Suggests the best offer angle  
- Generates a short outreach opener  
- Recommends the next action to take  

## Tech Used

- Python
- Requests + BeautifulSoup
- FastAPI (optional API)
- Command-line interface

## About

A practical lead-scoring tool built to help prioritize outreach and avoid wasting time on low-intent prospects.
